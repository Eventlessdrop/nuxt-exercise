# nuxt-exercise

## 事前に用意するもの
- node.js: v10.14.2
- npm: v6.4.1

## 環境構築
```
npm install
```
## 開発手順
- `npm run dev`
  - 開発サーバーを `localhost:3000` で起動します
  - ホットリローディング が効きます

## ディレクトリ構造
- nuxt.js の構造に準じています<br>
詳しくは、[nuxt.jsのドキュメント:ディレクトリ構造](https://ja.nuxtjs.org/guide/directory-structure/) をご確認ください
